TurboBlocks helps you disable unused CSS and JavaScript in WordPress, reduce page size, and improve performance — safely and without complex setup.

== Description ==

TurboBlocks – Smart Asset Loader is a lightweight WordPress performance plugin designed to eliminate unnecessary scripts and styles from your site.

Most WordPress plugins and themes load CSS and JavaScript on every page — even when not needed. This creates script bloat, slows down your site, and hurts Core Web Vitals.

TurboBlocks solves this by giving you simple, safe control over which assets load — helping you reduce page weight and improve speed instantly.

== Why Use TurboBlocks? ==

Most optimization plugins focus on caching and minification — but they still load unnecessary assets.

TurboBlocks goes further by allowing you to:

* Disable unused CSS and JavaScript
* Reduce page size and HTTP requests
* Improve Core Web Vitals (including INP)
* Optimize WooCommerce-heavy sites
* Fix performance issues without complex configuration

No complex rules. No risk. Just faster pages.

== Features ==

= Global Asset Control =
View all enqueued scripts and styles and disable the ones you don’t need — all from a single interface.

= WooCommerce Kill Switch =
Unload WooCommerce scripts and styles on non-shop pages like blog, homepage, and static pages.

= Safe Mode (Enabled by Default) =
Prevents removal of critical WordPress core assets to avoid breaking your site.

= One-Click Reset =
Instantly restore all settings to default if something goes wrong.

= Beginner-Friendly Interface =
Clean, simple UI designed for both developers and non-technical users.

== How It Works ==

TurboBlocks captures all scripts and styles loaded on your latest visited page and lets you selectively disable them.

1. Visit any page on your site
2. Open TurboBlocks settings
3. Disable unnecessary assets
4. Save — and instantly improve performance

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the **Plugins** screen in WordPress
3. Go to **Settings → TurboBlocks**
4. Visit any page on your site to detect assets
5. Disable unused scripts and styles

== Frequently Asked Questions ==

= Will this break my site? =
No. TurboBlocks includes Safe Mode which protects critical assets. You can also reset all settings anytime.

= Does it work with WooCommerce? =
Yes. The WooCommerce Kill Switch removes unnecessary Woo assets from non-shop pages.

= Does this plugin use external APIs? =
No. TurboBlocks works entirely within your WordPress installation and does not make external requests.

= Why don’t I see any assets in the list? =
TurboBlocks detects assets from your latest visited front-end page. Visit any page on your site first, then return to settings.
